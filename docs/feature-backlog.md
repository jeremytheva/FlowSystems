# Feature Backlog

The backlog tracks planned capabilities mapped to the unified module architecture. Items are grouped by module and tagged with priority.

## FlowBotCore
- [P0] Prompt guardrail configuration interface for ops administrators.
- [P1] Knowledge graph diff viewer with rollback support.
- [P2] Integration marketplace UI with status telemetry.

## BrandingFlow
- [P1] Persona-based tone recommendations connected to FlowBotCore prompts.
- [P2] Brand asset export wizard (SVG, PNG, PDF) integrated with design tokens.

## StrategyFlow
- [P0] Goal dependency visualisation with drag-and-drop editing.
- [P1] Automated risk scoring surfaced to GovernFlow dashboards.

## AttractFlow
- [P1] Campaign brief generator linked to WorkflowFlow automation triggers.
- [P2] Email template library with A/B testing stubs.

## ServeFlow
- [P0] Proposal approval workflow with audit trails stored in GovernFlow.
- [P1] Client portal document sharing integrated with EvolveFlow knowledge base.

## WorkflowFlow
- [P0] Automation canvas supporting conditional branches.
- [P1] SLA tracking with notifications into ServeFlow and GovernFlow.

## EvolveFlow
- [P1] Knowledge import wizard for Markdown and JSON sources.
- [P2] Community contribution queue with moderation states.

## GovernFlow
- [P0] Unified telemetry dashboard combining automation, adoption, and financial indicators.
- [P1] Governance log analytics with trend detection and anomaly surfacing.
